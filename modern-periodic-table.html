Modern Periodic Table

The modern periodic law states that the properties of elements are a periodic function of their atomic number — not atomic mass, which is where Mendeleev's original version occasionally broke down. Arranged this way, elements in the same column (group) share a valence electron structure, and predictable trends emerge as you move across periods and down groups.

Reactivity down Group 1 — the alkali metals

Li < Na < K < Rb < Cs

Reactivity increases down the group — the single outer electron sits farther from the nucleus each period, so it's held less tightly and gets lost more easily.

Reactivity down Group 17 — the halogens

F > Cl > Br > I

Reactivity decreases down the group — the opposite direction from the alkali metals, because halogens react by gaining an electron, and that pull weakens as atomic radius increases.

This is the core idea the modern table captures that earlier attempts (triads, octaves) only partially could: reactivity, atomic radius, ionization energy, and electronegativity all trend smoothly and predictably once elements are ordered by atomic number rather than mass.
